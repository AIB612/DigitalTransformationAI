## 🎯 业务问题

本模块解决三个核心问题：

1. **配送服务是否存在问题？** → 哪条线路时效差
2. **是否存在潜力销售区域？** → 哪里还有市场空间
3. **商品是否存在质量问题？** → 哪个产品退货多
